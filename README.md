## Mobility Access Developments for LA28 Olympics

*Los Angeles will host the 2028 Olympic and Paralympic Games. LA Metro and the LA28 organizing committee have framed the event as a "transit-first Olympics," tying it to the Twenty-Eight by '28 program: a package of 28 major transit projects—rail extensions, bus rapid transit, and connectivity hubs—targeted for completion before the Games.*

These investments include the D Line (Purple) subway extension to Westwood, the A Line Foothill extension toward Pomona, a new LAX transit center, and numerous bus and first-/last-mile improvements.
Despite this activity, a central question remains:

#### How much and where do these projects actually improve access to transit across Los Angeles County, particularly around 2028 Olympic venues?
The goal of this project is to provide a spatial, before-and-after assessment of transit accessibility using Geographic Information Systems (GIS), focusing on the change between the existing 2024/2025 network and the planned 2028 network.
